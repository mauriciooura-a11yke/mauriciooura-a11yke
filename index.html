Portal do Conhecimento | PPGs - EFLCH

O projeto foi implementado em abril de 2026 com o objetivo de organizar e disponibilizar, de forma estruturada, o conhecimento produzido pelos Programas de Pós-Graduação do campus Guarulhos. A solução utiliza dados provenientes de uma planilha Google, convertidos automaticamente para o formato JSON por meio do Google Apps Script com apenas um clique. Em seguida, os dados são enviados de forma automatizada para um repositório no GitHub e renderizados em interface web no formato de cards. Como principal diferencial, o projeto apresenta elevada facilidade de manutenção, além de oferecer rapidez no processamento e retorno dos filtros de busca, proporcionando uma experiência de navegação mais eficiente e dinâmica aos usuários.

Painel de Execução Orçamentária da Unifesp

Este projeto, implementado em agosto de 2024, utiliza Google Apps Script e Google Data Studio para automatizar o processamento e a visualização de dados orçamentários. A solução foi desenvolvida para realizar, sem intervenção humana, a extração, transformação e carga (ETL) de relatórios em Excel recebidos via Gmail a partir do sistema Tesouro Gerencial. Os arquivos são automaticamente convertidos em Google Planilhas, tratados e integrados ao painel de acompanhamento, cuja atualização ocorre por meio de acionadores automáticos programados. O painel tem como finalidade ampliar a transparência da execução orçamentária de um órgão federal, disponibilizando informações de forma estruturada e acessível. Entre os principais desafios enfrentados no desenvolvimento do projeto, destacam-se a integração com o Gmail para captura automática dos anexos, a conversão de arquivos Excel em Google Planilhas, o tratamento de tabelas com cabeçalhos compostos por células mescladas, a exclusão de linhas de totais e a delimitação automatizada do início e do fim das tabelas a serem processadas.

Painel de divulgação de Agenda de Bancas

O projeto foi implementado em maio de 2026 com o objetivo disponibilizar, de forma pública, no site do PPG em Educação, a sua agenda de bancas. Os dados são provenientes de uma planilha de controle que já era utilizada pela secretaria do PPGE. Os dados são atualizados automaticamente, conforme os agendamentos são validados na planilha. Dessa forma, os dados passaram a ser divulgados sem esforço adicional. Além de dar publicidade aos atos da secretaria, os alunos podem conferir a validação de suas solicitações de agendamento, assim como qualquer cidadão pode acompanhar as atividades do PPG.
