Portal do Conhecimento | PPGs - EFLCH

Implementei este projeto em abril de 2026 com o objetivo de organizar o conhecimento dos Programas de Pós-Graduação do campus Guarulhos. Os dados de uma planilha Google são convertidos para o formato JSON, via Google Apps Script, com um único clique. De forma automática, os dados são transferidos para o repositório no GitHub e renderizados em forma de cards.
